# awesome-dots

A dynamic tiling + floating awesomewm dots

Lockscreen and screenshot by [Nanoka](https://github.com/nhktmdzhg/)

![awesome](https://getbricked.github.io/awesome.png)

## Keybinds

### Launchers

| Key | Action |
|---|---|
| `Mod + Enter` | Open terminal (kitty) |
| `Mod + d` | Application launcher (rofi drun) |
| `Mod + Alt + v` | Clipboard manager (clipmenu via rofi) |
| `Mod + e` | File manager (thunar) |
| `Mod + b` | Browser (default browser) |
| `Mod + v` | Volume control (pavucontrol) |

### Window Management

| Key | Action |
|---|---|
| `Mod + f` | Toggle maximized |
| `Mod + Shift + f` | Toggle fullscreen |
| `Mod + q` | Close window gracefully |
| `Mod + Shift + q` | Force kill window (SIGKILL) |
| `Mod + space` | Toggle floating / centered |
| `Mod + Shift + space` | Cycle tiling layouts |

### Focus & Navigation

| Key | Action |
|---|---|
| `Mod + Tab` | Focus previous window |
| `Mod + ← ↑ → ↓` | Focus window in direction |
| `Mod + Escape` | Jump to previous tag |
| `Alt + Tab` | Cycle tags with windows |

### Tags (Workspaces)

| Key | Action |
|---|---|
| `Mod + 1-9` | Switch to tag N |
| `Mod + Shift + 1-9` | Move window to tag N and switch |
| `Mod + Ctrl + Shift + 1-9` | Toggle window on tag N |

### Layouts

| Key | Action |
|---|---|
| `Mod + Numpad 1-9` | Set layout by index |

### Resize & Master

| Key | Action (tiled) | Action (floating) |
|---|---|---|
| `Mod + Shift + ←` | Decrease master width | Resize window left |
| `Mod + Shift + →` | Increase master width | Resize window right |
| `Mod + Shift + ↑` | Increase master clients | Resize window up |
| `Mod + Shift + ↓` | Decrease master clients | Resize window down |

### Swap & Move Between Screens

| Key | Action |
|---|---|
| `Mod + Alt + ←/→` | Swap window with neighbor (cross-screen) |
| `Mod + Ctrl + ←/→` | Move window to adjacent tag/screen |

### Media & Hardware

| Key | Action |
|---|---|
| `XF86MonBrightnessUp/Down` | Brightness ±10% |
| `XF86AudioRaiseVolume` | Volume +5% |
| `XF86AudioLowerVolume` | Volume -5% |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioMicMute` | Toggle mic mute |
| `XF86AudioPlay/Pause` | Play/Pause (playerctl) |
| `XF86AudioPrev/Next` | Previous/Next track |
| `XF86AudioStop` | Stop playback |
| `XF86TouchpadToggle` | Disable touchpad |
| `XF86WLAN` | Toggle Wi-Fi |
| `XF86Bluetooth` | Restart Bluetooth |
| `XF86Sleep` | Suspend |

### Screen Capture

| Key | Action |
|---|---|
| `Mod + Shift + s` | Interactive screenshot (copied to clipboard) |

### System

| Key | Action |
|---|---|
| `Mod + Ctrl + r` | Restart AwesomeWM |
| `Mod + Ctrl + q` | Quit AwesomeWM |
| `Mod + l` | Lock screen |
| `Mod + n` | Nightmode (redshift -O 4500) |

### Mouse

| Action | Function |
|---|---|
| Right-click on desktop | Application launcher (rofi) |
| Scroll up/down on desktop | Previous/next tag |
| `Mod + Left click` | Move window |
| `Mod + Right click` | Resize window |
